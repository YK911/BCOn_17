# BCOn_17

Модуль 2:
- Синтаксис CSS: правило, селектор, свойства и значения.
- Подключение стилей: inline, embedded, linked
- CSS-селекторы: тега, класса, id, Х > Y ребенка, X Y потомка, атрибута
- Приоритетность. Каскад. Наследование.
- Цвет текста: свойство color
- Работа с цветом: rgb, hex, rgba
- Псевдоклассы состояния: :hover, :focus, :active

- Абсолютные единицы измерения: px
- Свойства: font-size, font-weight, font-style, font-family
- Свойства: line-height, letter-spacing, text-shadow

- Семейства шрифтов serif и sans-serif
- Свойства: text-align, text-decoration, text-transform, text-indent
- Псевдоэлементы ::firstLetter и ::selection
- Подключение внешних шрифтов через Google Fonts
